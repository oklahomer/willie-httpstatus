# willie-httpstatus
Module for willie to check given HTTP status code.

## Requirments
- Python
- [willie](http://willie.dftba.net/)

## Install

### Dependency
```
pip install willie

```

### willie-httpstatus
```
cd ~/.willie/modules/
git clone https://github.com/oklahomer/willie-httpstatus.git
ln -s willie-httpstatus/httpstatus.py ./
willie
```

## Usage
*`.httpstatus 200` returns 'OK'
